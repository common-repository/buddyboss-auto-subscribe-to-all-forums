== Description ==

Auto subscribe to all forums after user registration.
After enabling the buddyboss platform, Please make sure to configure the ff:
- Enable User registration
- Enable forums

If you have suggestions for a new add-on, feel free to email me at jcatama.ph@gmail.com

= Required Plugin =

[BuddyBoss Platform](https://www.buddyboss.com/platform/)

= Related PRO Add-ons =

[Buddyboss Social Group Subscription](https://gum.co/lHTvT)
Features:
- Automatically subscribe user to forum & all discussions after joining the Social Group
- Automatically subscribe user to new discussions
- Unsubscribe to forum & discussions when user leave the Social Group
- Do not auto subscribe users who opt-out from a specific discussion
- Hide main subscribe button from Social Group
- Compatible with Buddyboss + Learndash
